gformEZ

An AI-assisted Google Forms helper that reads a Google Form and generates suggested answers using Google's Gemini model.

Features

  • Reads Google Forms automatically
  • Extracts questions
  • Generates AI answers
  • Supports multiple-choice and text questions
  • User reviews answers before submission

Requirements

  • Python 3.11+
  • Playwright
  • Gemini API Key
  • Internet connection

Installation

  1. Clone the repository to your local directory.
  2. Install the required project dependencies.
  3. Install the Playwright browsers via your terminal.
  4. Obtain a Gemini API key from Google AI Studio.
  5. Run the execution script.

Usage

  1. Run python.py
  2. Enter your Gemini API key when prompted.
  3. Enter the Google Form URL.
  4. Wait for answers to generate.
  5. Review the generated answers.
  6. Fill and submit the form manually.

Limitations

  • AI answers may be incorrect.
  • Personal information must be entered manually.
  • Forms with unusual layouts may not be parsed correctly.
  • Requires a Gemini API key.
